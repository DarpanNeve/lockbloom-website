# LockBloom Website Context

## Overview

The LockBloom website is the production-grade marketing platform for the LockBloom password manager. It is built to be highly performant, SEO-optimized, and visually professional, serving as the primary source of organic traffic.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **Deployment**: Vercel (implied)

## Design Philosophy (APEX)

- **Professional aesthetic**: "Solid", high-contrast colors. No "AI-generated" neon gradients or generic assets.
- **Performance**: Perfect Lighthouse scores. Minimal JS.
- **Assets**: Use only verified assets (`logo.svg`, specific `screenshots/`).
- **Typography**: Inter / Space Grotesk.

## Core Components

- **Hero**: bold value proposition, "Get App" CTA, main app screenshot.
- **Features**: clear, icon-based breakdown of security/offline capabilities.
- **Security**: deep dive into encryption details (AES-256-GCM).
- **Showcase**: Visualizing the app interface.

## Assets

- Logos: `/public/logo.svg`
- Screenshots: `/public/screenshots/screenshot.png`

## SEO & ASO Strategy

- **Search**: JSON-LD Structured Data, heavily optimized metadata, dynamic sitemap/robots.
- **ASO Bridge**: Smart App Banners, strict manifest configuration for store linkage.
- **Traffic**: "LockBloom" brand dominance via accurate entity definition.
